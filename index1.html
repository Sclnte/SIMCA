SIMCA es una iniciativa tecnológica diseñada para modernizar y optimizar la agricultura tradicional.

Objetivos

Objetivo principal

Desarrollar un sistema automatizado para monitorear condiciones agrícolas en tiempo real y optimizar la toma de decisiones en cultivos mediante sensores y software de análisis.

Objetivos específicos

  • Diseñar una solución tecnológica que integre sensores de humedad, temperatura y luz.
  • Implementar una plataforma de monitoreo accesible vía web o aplicación móvil.
